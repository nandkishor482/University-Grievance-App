Project Overview
This is a full-featured web and mobile application designed to digitalize and automate student complaint management within a university ecosystem. It completely replaces traditional paper-based logs with an instant, real-time tracking system. The project features a dual-panel architecture with separate user flows for students and university authorities (HODs).

🚀 Key Features
Secure User Authentication: Integrated Firebase Authentication to provide a secure and private registration and login gateway for university students.

Real-Time Complaint Logging: Developed a seamless form where students can submit grievances. The app automatically maps and saves student IDs and email details to the backend.

Centralized NoSQL Database: Utilized Firebase Firestore to securely store data, manage application states, and process updates instantly.

Role-Based Admin Control Panel: Designed a dedicated dashboard for HODs and administrators to audit incoming complaints, check applicant profiles, and view issues in one place.

One-Tap Resolution Mechanism: Implemented a dynamic 'Resolve' toggle button within the Admin Panel, enabling authorities to instantly update ticket status and close complaints in real-time.
